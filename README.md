# htt*pdf*

A small HTTP microservice to generate PDFs.

[react-pdf](https://react-pdf.org/) + [Koa](https://koajs.com/) + [Docker](https://www.docker.com/)

## Usage

- Pull from [docker hub](https://hub.docker.com/r/n6g7/httpdf/)
- Follow [example](./example)
